Chatbot Widget Test Page

How to Test

  1. Click the chat bubble in the bottom-right corner
  2. Send a message to the chatbot
  3. Click "Talk to Human" to escalate to a support agent
  4. Check the Live Chat Dashboard for escalated conversations
  5. Respond from the dashboard and see real-time updates here

Test Scenarios

  1. Basic Chat: Ask "What are your store hours?"
  2. Product Question: Ask "Do you have any products on sale?"
  3. Escalation: Click "Talk to Human" and provide a reason
